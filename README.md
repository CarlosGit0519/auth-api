# Auth API

A REST API for user authentication, built as a backend portfolio project.

The project is being developed with a focus on maintainable code, security basics,
and clear API documentation.

## Planned stack

- Node.js
- TypeScript
- Express
- MongoDB
- Mongoose
- JSON Web Tokens (JWT)
- Bcrypt
- Swagger / OpenAPI
